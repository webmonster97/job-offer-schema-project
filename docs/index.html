Title

Type: string

Description: The job title.

Description

Type: string

Description: A detailed description of the job responsibilities and requirements.

Identifier

Type: object

Description: An object containing identification details for the job offer.
Properties:
  • Name: string - The name of the company offering the job.
  • Value: string - A unique identifier for the job offer.
  • SIRET: string - The company's SIRET number.
  • NAF Code: string - The NAF code of the company's main activity.

Date Posted

Type: string

Format: date-time
Description: The date and time when the job offer was posted.

Valid Through

Type: string

Format: date-time
Description: The date and time until the job offer is valid.

URL

Type: string

Format: uri
Description: The URL of the job posting.

Apply URL

Type: string

Format: uri
Description: The URL to apply for the job.

Employment Type

Type: string

Description: The type of employment offered.
Enum: FullTime, PartTime, Contract, Temporary, Internship, Other

Employment Level

Type: string

Description: The employment level.
Enum: none, senior, mid, junior

Hiring Organization

Type: object

Description: An object containing details about the hiring organization.
Properties:
  • Name: string - The name of the hiring organization.
  • Same As: string, uri - The URL of the organization's website.
  • Logo: string, uri - The URL of the organization's logo.

Recruiter

Type: object

Description: An object containing details about the recruiter.
Properties:
  • Name: string - The name of the recruiter.
  • Email: string, email - The recruiter's email.
  • Phone: string - The recruiter's phone number.

Job Location

Type: object

Description: An object containing details about the job location.
Properties:
  • Street Address: string - The street address of the job location.
  • Address Locality: string - The locality of the address.
  • Address Region: string - The region of the address.
  • Postal Code: string - The postal code of the address.
  • Address Country: string - The country of the address.
  • ISO Country: string - The ISO2 code of the country.

Remote Status

Type: string

Description: The remote work status of the job.
Enum: none, temporary, hybrid, fully

Base Salary

Type: object

Description: An object containing details about the base salary.
Properties:
  • Currency: string - The currency of the salary, e.g., USD, EUR.
  • Value: number - The amount of the salary.
  • Unit Text: string - The frequency of salary payment (Enum: Hour, Day, Week, Month, Year).

Qualifications

Type: string

Description: The qualifications required for the job.

Responsibilities

Type: string

Description: The responsibilities associated with the job.

Skills

Type: string

Description: The skills required for the job.

Incentive Compensation

Type: string

Description: Information about additional compensation, such as bonuses.

Industry

Type: string

Description: The industry of the job.

Work Hours

Type: string

Description: The work hours of the job.

Job Benefits

Type: array of strings

Description: The benefits offered by the employer.

Application Instructions

Type: string

Description: Instructions to apply for the job.

Non-Discrimination Policy

Type: string

Description: The employer's non-discrimination policy.

Data Protection Policy

Type: string

Description: Information about the data protection policy for applicants.
